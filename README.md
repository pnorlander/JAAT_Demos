# JAAT Demos

Demonstrations of [JAAT](https://github.com/Job-Ad-Research-at-QSB-LUC/JAAT) (Job Ad Analysis Toolkit) applied to different text sources. These notebooks show social scientists how JAAT's modules generalize beyond job advertisements to any text describing work activities.
